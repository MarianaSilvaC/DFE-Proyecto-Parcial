Proyecto de Gestión de Inventario para Farmacia

Desarrollé una aplicación web utilizando Python y Flask para una farmacia. Esta aplicación permitía realizar un CRUD (Crear, Leer, Actualizar, Eliminar) para llevar un registro preciso del inventario de materiales ópticos en la farmacia. El sistema facilitó la gestión de productos y mejoró la eficiencia en la administración de inventario, contribuyendo a un funcionamiento más eficaz de la farmacia.

Tecnologías utilizadas: Python, Flask, HTML, CSS, MySQL.

Puedes encontrar el código fuente en mi repositorio de GitHub: https://github.com/MarianaSilvaC/farmaciasPitic

Landing Page de Venta de Casas

Desarrollé una landing page utilizando HTML, CSS y SASS para promocionar la venta de casas de alta calidad. Esta página web presenta información detallada sobre las propiedades disponibles, incluyendo fotos, características y precios. Los visitantes pueden explorar fácilmente las opciones de vivienda y ponerse en contacto para obtener más información.

Tecnologías utilizadas: HTML, CSS, SASS.

Proyecto Espressoft - Sistema de Ventas para Cafetería

Espressoft es un software desarrollado por un equipo de alumnos de la Universidad de Sonora, en el cual participé en el área de análisis del programa y en el desarrollo de la aplicación. Este software es un sistema de ventas diseñado específicamente para cafeterías.

Proyecto de Gestión de Proyectos de Investigación

Desarrollé una aplicación web para la administración y gestión de proyectos de investigación en el Tecnológico de Estudios Superiores de Valle de Bravo durante un programa de verano de investigación. Utilicé las tecnologías Laravel y PHP para crear una plataforma que permitiera la evaluación y seguimiento de proyectos, mejorando la eficiencia en la toma de decisiones y la gestión de documentos requeridos.
